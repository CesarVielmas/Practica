El addSed o el paso de parametros o añadicion de datos , es algo bastante utilizado para poder pasar parametros predeterminados a la base de datos obviamente siendo un "Code First", de estos datos se le pasan a la base de datos y se guardan para tener unos datos como predeterminados dentro de esta misma,para ello se crea el AddSed , aunque se podrian añadir los datos manualmente en la base de datos , pero pues si se quiere hacer 100% "Code First" se hace de esta manera
